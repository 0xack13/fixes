# How to make a circular profile image with css

1. Create a class

```css
.circular-image{
  border-radius: 50%;
}
```

2. Use the class

```html
<img class="circular-image" src="<img class="smaller-image thick-green-border" src="https://bit.ly/fcc-relaxing-cat">
```
