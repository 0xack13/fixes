How to Update List of Available Ruby Versions on Linux
======================================================

Find Available versions
-----------------------

`rbenv install --list`

Install version
---------------

`rbenv install 2.1.5`

Update `ruby-build` as an rbenv plugin
--------------------------------------

`cd ~/.rbenv/plugins/ruby-build`

`git pull`
